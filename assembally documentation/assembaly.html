Syntax

Basic syntax structure:

LABEL: INSTRUCTION operand1, operand2
; Comments start with semicolon

Common instructions include MOV, ADD, SUB, JMP, and CALL.

Examples

Simple program example:

section .data
    msg db "Hello, Assembly!"

section .text
    global main
    main:
        MOV rax, 1
        RET
